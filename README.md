# Boletos

## Descrição

Api para validação de boletos bancarios.

## Instalação

Caso não tenha o yarn instalado

```bash
$ npm install --global yarn
```

```bash
$ yarn install
```

### Configurar o `.env`:

```bash
$ cp .env.example .env
```

### Rodar a aplicação:

```bash
$ yarn start:dev
```

## Rodando a aplicação

```bash
# development
$ yarn start:dev

# production
$ yarn start:prod
```
